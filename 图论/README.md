# 图论算法

图论是数学的一个分支，它以图为研究对象，研究顶点和边组成的图形的数学理论和方法。

## 📖 算法列表

- [广度优先搜索（BFS）](广度优先搜索/readme.md) - 按层遍历图或树
- [深度优先搜索（DFS）](深度优先搜索/readme.md) - 深度优先遍历图或树

## 🔍 算法特点

### 广度优先搜索（BFS）
- 适用于寻找最短路径
- 按层序遍历
- 使用队列实现

### 深度优先搜索（DFS）
- 适用于深度探索
- 使用栈或递归实现
- 适合回溯问题

## 📚 学习建议

1. 先理解图的基本概念（顶点、边、度等）
2. 掌握图的存储方式（邻接矩阵、邻接表）
3. 学习BFS和DFS的基本思想
4. 通过练习加深理解

## 🎯 应用场景

- 社交网络分析
- 路径规划
- 网络流问题
- 最小生成树
- 最短路径
